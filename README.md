# Excel-Reports
📁 Project Overview
This repository contains an Excel dataset focused on global economic data, specifically GDP (Gross Domestic Product) in constant local currency units (LCU) across multiple countries and years.
The dataset is structured to support data analysis, visualization, and economic trend studies, making it suitable for academic projects, data analytics portfolios, and business intelligence use cases.

📂 File Included

QUESTION_EXCELREPORTS (2) (4).xlsx


🗂️ Dataset Structure
The Excel file contains multiple sheets, each serving a specific purpose:
1. 🌍 Metadata – Countries
Provides country-level classification details:

Country Code
Country Name
Region (e.g., South Asia, Europe & Central Asia)
Income Group (e.g., Low income, High income)

👉 Useful for:

Data grouping
Filtering by region or income category
Dimensional modeling in BI tools


2. 💰 GDP (Constant LCU)
Contains economic data across years:

Country Code
Indicator Name (GDP - constant LCU)
Year-wise GDP values (1960–2023)

👉 Key Features:

Longitudinal (time-series) data
Country-wise economic trends
Enables historical growth analysis


🔍 Use Cases
This dataset can be used for:

📈 Trend Analysis – Study GDP growth over decades
🌎 Regional Comparison – Compare economies across regions
📊 Dashboard Creation – Power BI / Tableau visualization
🧪 Data Science Projects – Forecasting, regression models
🎓 Academic Research – Economic development studies

📌 Notes

GDP values are in constant LCU, meaning inflation-adjusted values.
Some entries may have missing data for certain years.
Regions and income categories follow standard global classifications.
